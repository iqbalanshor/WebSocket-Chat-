## 1. Setup Lingkungan Pengujian
### • Instal Node.js: Download dan instal Node.js dari https://nodejs.org
### • Instal Pustaka WebSocket: Jalankan perintah berikut untuk menginstal pustaka WebSocket:
```bash
npm install ws

npm init -y

npm install express ws
```
